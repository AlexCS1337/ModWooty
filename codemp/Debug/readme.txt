ModWooty .21 -- 1/29/04

A melee-oriented serverside only mod for JK: Academy 1.01.

Installation:

Drag'n'drop this folder inside your \GameData directory. 

To start a ModWooty server, run jampded with an additional switch: "+set fs_game modwooty"

Documentation:

Type /help in a player console for user documentation. No modification to your autoexec or server.cfg are necessary.
